# 🏠 집덕(ZipDuck) --- "청약을 가장 잘 아는 친구, 집덕."

> **복잡한 청약, 이제 AI가 '덕질'해서 알려드립니다.**\
> 집덕은 청약 제도·분양 공고·자격 조건을 AI가 분석해\
> **가장 빠르고 정확한 청약 추천**을 제공하는 주거 정보 서비스입니다.

## ✨ 소개 (Introduction)

**집덕(ZipDuck)** 은\
청약 제도에 익숙하지 않은 사용자도 손쉽게 정보를 얻을 수 있도록 만든\
**AI 기반 청약 추천·진단·데이터 분석 플랫폼**입니다.

집덕은 다음을 목표로 합니다:

-   공공데이터 기반으로 **신뢰도 높은 청약 정보**를 제공
-   사용자의 조건을 자동 분석하여 **개인화된 추천** 제시
-   관심 지역·연령·가구 구성 등을 바탕으로 **청약 가능성 판단**
-   지역별·유형별 **데이터 기반 통계 리포트 제공**
-   친근한 브랜드 캐릭터 **집덕이**가 사용자 경험을 돕는 서비스
